1. position: static (default):
This box is static. It follows the normal document flow and cannot be offset.
Static is the default position for all elements. No special positioning is applied.
2. position: relative:
This box is relative. It is offset left: 40px and top: 10px from its normal position.
Relative positioning moves the element from its normal flow position, but space is still reserved for it.
3. position: absolute (inside a relative container):
This box is absolute. It is positioned left: 20px, top: 20px inside its relative parent.
The parent container is position: relative. Absolute elements are removed from normal flow.
4. position: fixed (viewport):
Scroll the page to see the fixed box stay in the corner.
This box is fixed.
It stays in the bottom-right corner of the viewport.
5. position: sticky (sticks while scrolling):
This box is sticky.
It sticks to the top of its scrollable container when you scroll down.
Scroll this container.
The sticky box will remain visible at the top until you scroll past its parent.
